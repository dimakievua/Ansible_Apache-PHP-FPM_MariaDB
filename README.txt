Ansible playbook to install Apache + PHP-FPM on one host and MariaDB on another.Basic site is copying to Apache host, virtual host created with rewrite and PHP-FPM proxy settings. There is possibility to manage PHP-FPM, Apache configuratino file via JSON template. Variables are stored in global_vars.
Maria DB isntalled and configured with example DB. 

All setup are prepared for Ubuntu/Debian, there is no RedHat/CentOS support yet.

Your comments are welcome.
